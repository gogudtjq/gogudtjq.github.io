연산자 operator

+ - * / % ++ –

증가연산 ex) n++; , ++n;
감소연산 ex) n–; , – n;

대입의 우선순위
ex) int num1 = 5
int num2 = num1 ++; //이럴경우 증가연산보다 대입이 먼저 진행된다.

대입 연산자

기호 ex
= ex)val = 20;
+ ex)val = 4+3;
- ex)val = 4-3;
* ex)val = 4*3;
/ ex)val = 4/3;
% ex)val = 4%3;

% : 왼쪽의 피연산자 값을 오른쪽의 피연산자 값으로 나눴을 때 얻게 되는 나머지를 반환한다.
