String

ex) String 변수명 = “ 문자열 “ ;

문자열이 직접 변수에 저장되는 것이 아니고,
문자열은 String 객체로 생성되고
변수는 String 객체를 참조합니다

String 타입 메소드

indexOf 문자열 내에서 주어진 문자열의 위치를 리턴

toLowerCase 알파벳 소문자를 변환한 문자열 리턴

toString 문자를 출력

trim 앞 뒤 공백을 제거한 새로운 문자열을 리턴
